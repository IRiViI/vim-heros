# Vim Heroes: Programmers

## Why this game?
I use AI more and more for everything I code. Which means I'm losing street cred.
So I decided to learn Vim motions to compensate.
But the problem is that I don't code anymore, so I couldn't learn Vim on my own projects.
So the next logical step was to vibe code a game to learn Vim.

## How to play

Make sure you have [Rust](https://www.rust-lang.org/tools/install) installed, then:

```bash
git clone https://github.com/<your-username>/vim-heros.git
cd vim-heros
cargo run
```

## Inspiration
Inspired by [vim-be-good](https://github.com/ThePrimeagen/vim-be-good), but that required me to install Nvim and that is not VS Code, so I couldn't play it.
